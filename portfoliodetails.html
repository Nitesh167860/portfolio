Professional Portfolio Website

Project Overview

Professional Portfolio Website is a modern and fully responsive web application developed using HTML5, CSS3 and JavaScript. It showcases my skills, projects, education, internship experience and contact information with a visually appealing user interface. The website includes Dark & Light Mode, Smooth Animations, Floating Bubble Effects, Water Liquid Style Profile Image, Custom Snack Style Cursor, Project Showcase Section, Responsive Design and Contact Section.
